# Projects

Runtime project files are written here.

This directory is ignored by git because it can contain generated scripts, images, audio, captions, and rendered videos.

Typical structure:

```text
projects/<project-id>/
  project.json
  source.md
  segments.json
  script.json
  visual-bible.json
  render-plan.json
  storyboards/
  storyboards.json
  voiceover.wav
  captions.vtt
  timings.json
  renders/final.mp4
```
